============
Installation
============
Installation from source_,
working directory where source is checked out

.. code-block:: bash
  
    $ pip install .

This package will be soon available on *PYPI*, allowing

.. code-block:: bash
  
    $ pip install combat-stadis

=====
Usage
=====

The program run entirely in a jupyter-notebook, thus it is aslo a requirement.

.. code-block:: bash
  
    $ jupyter-notebook
